 30: /**
 31:  * Provides a base for all JSecurity Hash algorithms with support for salts and multiple hash iterations.
 32:  *
 33:  * <p>Read <a href="http://www.owasp.org/index.php/Hashing_Java" target="blank">http://www.owasp.org/index.php/Hashing_Java</a> for a
 34:  * good article on the benefits of hashing, including what a 'salt' is as well as why it and multiple hash iterations
 35:  * can be useful.
 36:  *
 37:  * <p>This class and its subclasses support hashing with additional capabilities of salting and multiple iterations via
 38:  * overloaded constructors</p>.
 39:  *
 40:  * @author Les Hazlewood
 41:  * @since 0.9
 42:  */

 53:     /**
 54:      * Creates an new instance without any of its properties set (no hashing is performed).
 55:      *
 56:      * <p>Because all constructors in this class
 57:      * (except this one) hash the <tt>source</tt> constructor argument, this default, no-arg constructor is useful in
 58:      * scenarios whenyou have a byte array that you know is already hashed and just want to set the bytes in their
 59:      * raw form directly on an instance.  After instantiating the instance with this default, no-arg constructor, you
 60:      * can then immediately call {@link #setBytes setBytes} to have a fully-initiallized instance.
 61:      */

100:     /**
101:      * Creates a hash of the specified <tt>source</tt> using the given <tt>salt</tt> a total of
102:      * <tt>hashIterations</tt> times.
103:      *
104:      * <p>By default, this class only supports Object method arguments of
105:      * type <tt>byte[]</tt>, <tt>char[]</tt> and <tt>String</tt>.  If either argument is anything other than these
106:      * types a {@link org.jsecurity.codec.CodecException CodecException} will be thrown.
107:      *
108:      * <p>If you want to be able to hash other object types, or use other salt types, you need to override the
109:      * {@link #toBytes(Object) toBytes(Object)} method to support those specific types.  Your other option is to
110:      * convert your arguments to one of the default three supported types first before passing them in to this
111:      * constructor</tt>.
112:      *
113:      * @param source         the source object to be hashed.
114:      * @param salt           the salt to use for the hash
115:      * @param hashIterations the number of times the <tt>source</tt> argument hashed for attack resiliency.
116:      * @throws CodecException if either Object constructor argument cannot be converted into a byte array.
117:      */
118:     public AbstractHash(Object source, Object salt, int hashIterations) throws CodecException {
119:         byte[] sourceBytes = toBytes(source);
120:         byte[] saltBytes = null;
121:         if (salt != null) {
122:             saltBytes = toBytes(salt);
123:         }
124:         byte[] hashedBytes = hash(sourceBytes, saltBytes, hashIterations);
125:         setBytes(hashedBytes);
126:     }

182:      * @param bytes the bytes to hash
183:      * @param salt  the salt to use for the initial hash
184:      * @return the hashed bytes

190:     /**
191:      * Hashes the specified byte array using the given <tt>salt</tt> for the specified number of iterations.
192:      *
193:      * @param bytes          the bytes to hash
194:      * @param salt           the salt to use for the initial hash
195:      * @param hashIterations the number of times the the <tt>bytes</tt> will be hashed (for attack resiliency).
196:      * @return the hashed bytes.
197:      */
198:     protected byte[] hash(byte[] bytes, byte[] salt, int hashIterations) {
199:         MessageDigest md = getDigest(getAlgorithmName());
200:         if (salt != null) {
201:             md.reset();
202:             md.update(salt);
203:         }
204:         byte[] hashed = md.digest(bytes);
205:         int iterations = hashIterations - 1; //already hashed once above
206:         //iterate remaining number:
207:         for (int i = 0; i < iterations; i++) {
208:             md.reset();
209:             hashed = md.digest(hashed);
210:         }
211:         return hashed;
212:     }

219:      * next time this method is called.

221:      * @return a hex-encoded string of the underlying {@link #getBytes byte array}.
222:      */
